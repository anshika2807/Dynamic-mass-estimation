# Dynamic-mass-estimation
This project focuses on estimating the dynamical mass of a galaxy cluster using real astronomical data from the Sloan Digital Sky Survey (SDSS).

The analysis applies astrophysical models to study galaxy motion and provides strong evidence for the existence of dark matter.

🎯 Objectives
Analyze galaxy cluster data from SDSS
Calculate velocity dispersion
Apply the Virial Theorem
Compare dynamical vs luminous mass
Validate presence of dark matter
🛠️ Tools & Technologies
Python
NumPy
Pandas
Matplotlib
Astropy
📊 Methodology
1. Data Collection
Retrieved spectroscopic data (redshift, spatial position) from SDSS
2. Data Preprocessing
Cleaned dataset using Pandas
Removed noise and outliers
3. Feature Engineering
Computed:
Redshift
Radial velocity
Velocity dispersion
4. Mass Estimation
Used Virial Theorem:
M ∝ (σ² × R) / G
5. Visualization
Galaxy distribution plots
Velocity histograms
Cluster structure mapping
📈 Results
Estimated dynamical mass successfully

Found:

Dynamical mass is significantly greater than luminous mass

➡️ Strong evidence for dark matter presence

🔍 Key Insights
Galaxy clusters are dominated by unseen mass
Velocity dispersion reveals gravitational structure
Virial Theorem works effectively at cosmic scale
